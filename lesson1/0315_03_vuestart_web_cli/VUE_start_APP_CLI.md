

# 뷰 시작하기 앱 CLI 프로젝트 순서
1. 프로젝트 폴더 만들기
2. Git bash 터미널 => 뷰 CLI 설치하기
  ```bash
  npm i -g @vue/cli
  ````
3. 프로젝트 생성 하기
```bash
  vue create app
```
4. 프로젝트 실행
```bash
  ls -la
  cd app
  npm run serve
```
5. 프로젝트 구조 확인, 수정하기
6. 프로젝트 빌드 하기
```bash
npm run build
```
7. 프로젝트 빌드(dist) 내용 수정 하기
8. 닷홈에 배포하기
9. 웹사이트 띄우기
